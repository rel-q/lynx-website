## 属性

### `data`

{' '}

<AndroidOnly /> <IOSOnly /> <VersionBadge>3.4</VersionBadge>

```tsx
data?: Record<string, unknown>;
```

向 frame 内嵌的 Lynx 页面传递数据。

### `global-props`

{' '}

<AndroidOnly /> <IOSOnly /> <VersionBadge>3.6</VersionBadge>

```tsx
'global-props'?: Record<string, unknown>;
```

向 frame 内嵌的 Lynx 页面传递 `globalProps`, 内嵌页面可以通过[`lynx.__globalProps`](../../lynx-api/lynx/lynx-global-props.mdx) 进行读取。

### `src`

{' '}

<AndroidOnly /> <IOSOnly /> <VersionBadge>3.4</VersionBadge>

```tsx
src: string;
```

设置 frame 资源的加载路径。

## 事件

前端可以绑定相应的事件回调来监听元素的运行时行为，如下所示。

### `bindload`

{' '}

<AndroidOnly /> <IOSOnly /> <VersionBadge>3.6</VersionBadge>

```tsx
bindload = (e: FrameLoadEvent) => {};
```

| 字段          | 类型   | 可选 | 默认值 | 平台                                                         | 起始版本 | 描述                 |
| ------------- | ------ | ---- | ------ | ------------------------------------------------------------ | -------- | -------------------- |
| statusCode    | number | 否   | –      | <AndroidOnly /> <IOSOnly /> <VersionBadge>3.6</VersionBadge> | 3.6      | Frame 加载状态码。   |
| statusMessage | string | 否   | –      | <AndroidOnly /> <IOSOnly /> <VersionBadge>3.6</VersionBadge> | 3.6      | Frame 加载状态信息。 |
| url           | string | 否   | –      | <AndroidOnly /> <IOSOnly /> <VersionBadge>3.6</VersionBadge> | 3.6      | Frame 加载的 url。   |

绑定 frame 加载事件回调。
