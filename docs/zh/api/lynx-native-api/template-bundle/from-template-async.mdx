# fromTemplateAsync

基于模板二进制数据异步创建一个 TemplateBundle。该方法可避免阻塞主线程。

## 语法

### Harmony

```typescript
public static async fromTemplateAsync(template: ArrayBuffer): Promise<TemplateBundle>;
```

#### 参数说明

- `template`：`ArrayBuffer`，模板二进制数据的缓冲区。

#### 返回值
- 返回一个 `Promise`，resolve 后得到新的 `TemplateBundle` 对象。
