# onReceivedError

收到后台运行时中的错误时调用，错误可能来自 JS 或 Native（C++）。

## 语法

### Android

```java
public void onReceivedError(LynxError error);
```

#### 参数

- `error`: 错误信息。

### iOS

```objc
- (void)runtime:(LynxBackgroundRuntime *)runtime didReceiveError:(NSError *)error;
```

#### 参数

- `error`: 错误信息。
