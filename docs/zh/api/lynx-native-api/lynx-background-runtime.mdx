# LynxBackgroundRuntime

LynxBackgroundRuntime 是 Lynx 的后台线程运行时，负责执行不直接影响屏幕像素渲染的脚本逻辑，例如业务数据处理、网络请求以及 Native Module 的调用。它与 `LynxView` 处于同一层级，二者各自独立：`LynxView` 负责 UI 渲染，`LynxBackgroundRuntime` 负责非渲染的脚本执行。

## 适用场景

- 运行不涉及像素渲染的业务逻辑
- 在非 UI 线程处理网络与 Native Module 调用
- 与 `LynxView` 并行工作，互不影响渲染
