---
title: profileEnd()
---

标记最近的 [trace event](https://perfetto.dev/docs/instrumentation/track-events) 的结束。

## 语法

```typescript
profileEnd(): void;
```

## 示例

```typescript
// 结束最近的 trace event
lynx.performance.profileEnd();
```
