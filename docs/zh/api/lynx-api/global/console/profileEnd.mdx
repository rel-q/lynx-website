# console: profileEnd() method

`profileEnd` 方法用于结束记录性能。

## 语法

```ts
profileEnd(profileEndName: string): undefined;
```

### 参数

#### profileEndName

需要结束的 profile 的名称。

### 返回值

无（`undefined`）。
